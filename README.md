# grzybstagram

Niezależny blog o grzybach w kontekście zdrowia. Fakty, źródła, nauka i opinie bez cenzury.

Stack: Astro 6 + Tailwind CSS + MDX.

## Lokalnie

```bash
npm install
npm run dev   # http://localhost:4321
```
